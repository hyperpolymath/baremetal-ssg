== Overview

Baremetal SSG is a satellite project in the https://github.com/hyperpolymath/poly-ssg-mcp[poly-ssg-mcp] network. It provides high-performance, minimal-abstraction adapters for static site generators written in functional, systems, and scientific programming languages.

=== Key Features

* *28 SSG Adapters* — Comprehensive coverage of polyglot SSGs
* *18 Languages* — Rust, Haskell, Elixir, Julia, Clojure, and more
* *MCP Integration* — Seamless connection to poly-ssg-mcp hub
* *RSR Compliant* — Follows Rhodium Standard Repository guidelines
* *Deno Runtime* — Modern, secure JavaScript/TypeScript runtime

== Quick Start

[source,bash]
----
# Clone the repository
git clone https://github.com/hyperpolymath/baremetal-ssg.git
cd baremetal-ssg

# Install dependencies
just deps

# List available adapters
just list

# Run tests
just test
----

=== Using an Adapter

[source,javascript]
----
import * as zola from "./adapters/zola.js";

// Connect to the SSG
await zola.connect();

// Initialize a new site
await zola.tools[0].execute({ path: "./my-site" });

// Build the site
await zola.tools[1].execute({ path: "./my-site" });
----

2. Implement the adapter interface
3. Add tests in `tests/adapters/`
